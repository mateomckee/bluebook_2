Newbook Tips

1. Search for key terms separated by a space

2. You can search by course name, course label (e.g. WRC), section number, professor name, semester, and year

3. If there is a "" icon in a search result, that means the course has evaluation data available (ratings)

4. To get the best results, be specific!
